# M-Pan云盘
简介：一个基于SpringBoot的存储系统，类似百度网盘

## 功能模块

1. 用户模块
2. 文件模块
3. 回收站模块
4. 分享模块

### 用户模块业务点

- 登录
- 注册
- 忘记密码并修改密码
- 退出登录
- 知道密码并修改密码
- 用户搜索历史的维护

### 文件模块业务点

- 上传文件
- 秒传
- 断点续传
- 新建文件夹
- 下载文件
- 删除文件
- 重命名文件
- 分享文件
- 复制文件
- 移动文件
- 预览文件
- 搜索文件

### 回收站模块业务点

- 清空回收站
- 还原回收站

### 分享模块业务点

- 分享文件
- 查看已分享文件
- 取消分享
- 文件的转存

## 模块设计

技术模块与业务模块解耦，**插件式**集成技术模块

### 版本说明

JDK 1.8

SpringBoot 2.1.9.RELEASE

Swagger2 2.8.0

Mybatis-Plus 3.3.2

Redis 7.2.4

Mysql 8.0

RocketMQ 4.5.1
